# 个人记账管理系统 python284

## 项目概述

个人记账管理系统是基于 Python Django 框架开发的一款网络应用，旨在帮助用户高效管理个人财务，记录日常收入与支出，并提供统计功能以便用户清晰掌握财务状况。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：HTML

## 功能模块

- **用户认证**
  - 用户登录
- **个人中心**
  - 个人信息修改
- **记账管理**
  - 收入记录
  - 支出记录
  - 收入记账管理
  - 支出记账管理
- **统计功能**
  - 收入统计
  - 支出统计

## 系统角色

- 普通用户：可登录系统，管理个人账目。

## 运行环境

- Python 环境推荐版本：3.6+
- Django 版本：2.2+
- MySQL 版本：5.7+

## 部署步骤

1. 环境准备：安装 Python、MySQL 和相关依赖。
2. 数据库配置：根据项目提供的数据库文件初始化 MySQL 数据库。
3. 项目配置：配置 Django 项目设置，包括数据库连接等。
4. 运行项目：使用 Django 提供的管理命令启动开发服务器。
